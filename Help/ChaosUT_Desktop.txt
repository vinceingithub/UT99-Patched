ChaosUT Desktop UMOD
====================

This is an optional add-on for the Chaos UT Mod.
It installs a desktop that replaces the one with
the text 'Unreal Tournament' and places the 
words 'Chaos UT' there.  Enjoy!


Files Included
==============
CUT-Logo-Map.unr
CUT_Desktop.utx
